<a name="Create"></a>
## Create

創建資料庫：`CREATE DATABASE <db name>;`

```mysql
mysql> CREATE DATABASE `TUTORIALS`;
Query OK, 1 row affected (0.00 sec)
```

<a name="Select"></a>
## Select

選擇資料庫：`USE <db name>;`

```mysql
mysql> USE `TUTORIALS`;
Database changed
```

<a name="Drop"></a>
## Drop

刪除資料庫：`DROP DATABASE <db name>;`

```mysql
mysql> DROP DATABASE `TUTORIALS`;
Query OK, 0 rows affected (0.00 sec)
```
